Minigames

Multiple C++ Minigames, developed raylib library.

About

Minigames developed while taking the course: C++ Fundamentals: Game Programming For Beginners. The course explained basic C++ syntax: variables, methods, loops, classes, inheritance, pointers. It also, mentioned how to use external libraries, such as Raylib, and how to work with Vectors.

Collision

This was the introductory project. The user can move the square left and right with the A and D keys. When there is a collision with the circle, the game is over.

  • This section of the course focused on how to use external libraries.
  • First concepts of programming, like data types and variables.
  • Covered how to use the functions from the raylib library to draw basic shapes, and move them around the screen, while also checking for collisions.

Jumping

In this game, the character has to jump through an array of hazards, when it arrives at the finish line, the user win, but if they collide with a hazard, the game is over.

  • The course introduced animation concepts, how to import sprite sheets, and how to animate them an be frame-rate independant.
  • Worked with Custom Data Types (structs), and arrays.
  • Created a win-lose situation.

Classy Clash

In this game, the character is chased by two enemies, the user has to use the WASD keys to control the character around the world, and attack the monsters to destroy them.

  • Used object oriented programming.
  • Used class inheritance.
  • Learned about pointers.
  • Used virtual functions to override in child class.
  • Implemented a basic health system.
